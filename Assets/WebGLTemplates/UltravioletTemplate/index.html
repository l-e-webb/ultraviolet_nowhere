About

Ultraviolet Nowhere is a game about searching space for distant transmissions that may be drifting between the stars. Use the arrow keys to aim your telescope. When you find a transmission, switch frequency bands using the console until you have the strongest signal, and home in on it with the fine-tuning slider. Some transmissions have text, some are audio only.

Ultraviolet Nowhere was made at the Enthusiasm Collective site in Portland, Oregon for Global Game Jam 2018. Check out the jam page here, or view the latest source code here.
